# --- Multi-agent Deep Mode (Topic 5.5) ---
# NOTE: crewai pulls in ~50+ transitive deps (chromadb, langchain-*,
# opentelemetry, etc). Streamlit Cloud build takes 3-5 minutes.
#
# chromadb constraint MUST match CrewAI's own upper bound.
# CrewAI's dependency spec: `chromadb~=1.1.0` (i.e., >=1.1.0, <1.2.0).
# Using `chromadb>=1.1.2` alone creates a false conflict because pip
# interprets it as potentially wanting 1.2.x+. Matching CrewAI's
# ~=1.1.0 range avoids the conflict AND ensures pydantic-settings
# 2.10+ compatibility (chromadb 1.1.x fixed the earlier
# chroma_server_nofile type-inference issue).
#
# If runtime "chroma_server_nofile" error still appears on cloud
# despite this pin, the CREWAI_AVAILABLE flag in app.py (paired with
# the disabled Deep Mode checkbox) provides graceful degradation.
crewai>=1.10.0
chromadb~=1.1.0
